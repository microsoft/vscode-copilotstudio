# Common

This directory contains more than only Contracts and Data classes. 
It also contain common helpers and utilities that are shared across projects.
As the content of this directory grow, it is expected that a new project may be cut out of it.

We should still try to follow [architecture and design principles](../../../../docs/LanguageServerArchitecture.md) as much as possible. Which means it's preferable to share implementation details through Dependency Injection whenever possible.
