# Support

## How to file issues and get help

The Copilot Studio extension for VS Code uses
[GitHub Issues](https://github.com/microsoft/vscode-copilotstudio/issues) to
track bugs and feature requests. Please search
[existing issues](https://github.com/microsoft/vscode-copilotstudio/issues)
before filing a new one to avoid duplicates.

### Recommended: use the built-in issue reporter

The extension includes a custom issue-reporting command that automatically
attaches diagnostic information (such as your session ID) to help us
investigate problems more quickly:

1. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Run **Help: Copilot Studio: Report Issue**
3. Fill out the form and submit.

### Scope

This repository covers the **VS Code extension** and its language server. For
help with Microsoft Copilot Studio itself (the service, the web authoring
experience, or agent behavior), please use:

- [Microsoft Copilot Studio documentation](https://learn.microsoft.com/microsoft-copilot-studio/)
- [Microsoft Copilot Studio Community forums](https://community.powerplatform.com/forums/thread/home)

## Microsoft Support Policy

Support for this project is limited to the resources listed above. This is an
open-source project maintained by Microsoft — response times for issues and
pull requests may vary.
